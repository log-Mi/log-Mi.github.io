Posted on 2021-07-26 Edited on 2021-08-01

前端学习路径

  • HTML 标签

  • CSS 盒模型、定位、文档流、优先级

  • JavaScript DOM基本操作(变量、函数、事件、this、传参、for、if else、获得dom方法)–写出一些基本页面交互（弹出菜单、轮播图、折叠、DOM节点crud）

  • Node.js 前端工程化 express设静态目录、创建服务、响应get请求（基本要求：安装npm包，创建服务、响应请求）

  • Ajax 判断readyState和status，open/send（返回json/js在后端获得数据创建dom）

  • HTTP+TCP/IP（计算机网络相关）

  • 原生Js值/引用传递

  • Vue.js 数据驱动视图、模块化、组件化、单向数据流、双向绑定（先在vue脚手架自带的hello world页面上实现前述基本DOM操作，然后npm安装axios请求之前的get接口返回json渲染生成页面）熟悉vue命令

  • Vue 项目，跟着写，积累代码量

  • 深入原生Js、ES6、new、闭包、原型、继承、call/aplly、原型链、作用域链等
